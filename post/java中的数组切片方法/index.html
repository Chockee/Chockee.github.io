Java中的数组切片方法

Written by ZhuJ1ntao with ♥ on April 3, 2020 in 练级之路
  • 调用Java.util.Arrays中的copyOfRange()方法可以实现对数组的切片。

  • 具体参数为：

    Type[] copyOfRange(Type[] array, int start, int end);

  • 其中array为原来的数组，start为切片的起始位置，end为切片终止位置的前一位(取不到)：[start, end）
